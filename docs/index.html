How to reuse

  1. Put your study_area.shp (+ sidecars) and presence.csv under /data.
  2. Add GeoTIFFs to /data/rasters (any number, any names).
  3. Set parameters in the Rmd YAML (EPSG, background size, output names).
  4. Knit to HTML and (optionally) publish via GitHub Pages.
